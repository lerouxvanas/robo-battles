## How to test

1. `npm run build` — confirm `dist/sw.js` and `dist/manifest.webmanifest` are present
2. `npm run preview` — open Chrome DevTools → Application → Service Workers: confirm SW registered
3. DevTools → Application → Manifest: confirm no validation errors, icons load, `start_url` correct
4. DevTools → Network → Offline → reload: confirm app loads without network
5. On iPhone: visit deployed GitHub Pages URL → Safari share → Add to Home Screen → confirm name + icon appear
6. Launch from home screen: confirm standalone mode (no browser chrome)
